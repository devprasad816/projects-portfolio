# Electronics Sales Analysis (2019)

## Objective
Analyze 2019 electronics sales data to identify revenue trends,
top-performing products, and time-based sales patterns.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Power BI

## Project Structure
- **Python/** – Data cleaning, EDA, and visualizations
- **SQL/** – Query logic and aggregations
- **PowerBI/** – Interactive dashboard
- **Project_Document.pdf** – Detailed project report

## Key Insights
- Peak sales observed during holiday months
- Laptops and phones generated the highest revenue
- Strong correlation between order time and purchase volume
